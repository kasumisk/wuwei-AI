二、AI Prompt 设计（你的核心壁垒）

这是你最重要的部分，我直接给你一套可用结构

⸻

🧠 核心 Prompt（简化版）

你是一个专业减脂饮食教练。

你的目标不是提供营养知识，而是帮助用户做“吃或不吃”的决策。

请根据用户输入的食物，输出：

1. 是否建议食用（YES / NO / LIMITED）
2. 原因（一句话，简单易懂）
3. 建议（如何调整）
4. 补救措施（如果吃了怎么办）

要求：
- 不要长篇解释
- 用口语
- 像朋友一样
- 有一点“严格但不攻击人”

用户信息：
- 性别：男
- 目标：减脂
- 当前状态：正常饮食

用户输入：
{食物内容}


⸻

🧪 示例输出（理想效果）

{
  "decision": "LIMITED",
  "reason": "油脂和糖都偏高，容易超标",
  "suggestion": "减少一半分量，优先吃蛋白质",
  "compensation": "今天多走30分钟或减少晚餐碳水"
}


⸻

⭐ 进阶（你后面可以做）
	•	个性化 prompt（根据用户习惯）
	•	情绪识别（暴食 / 焦虑）
	•	长期饮食策略（不是单次判断）

⸻

三、冷启动增长方案（非常关键）

你这个产品不靠广告是能起量的，但要方法对。

⸻

🎯 核心策略：内容驱动 + 工具转化

⸻

1️⃣ 小红书 / 抖音（主阵地）

内容类型：

类型1：对比类（爆款）

“这份外卖 = 跑步2小时？”

类型2：打脸类

“你以为健康，其实最胖的食物”

类型3：挑战类

“我用AI减肥7天，结果…”


⸻

👉 关键：

不要推广App

⸻

2️⃣ 转化路径（很关键）

内容 → 评论区引导 → 小程序 / H5

例如：

评论「饮食」领取AI分析工具

⸻

3️⃣ 冷启动最小闭环

你可以这样做：
	1.	做一个极简H5（不用App）
	2.	用户上传图片 → 返回AI结果
	3.	收集用户数据

⸻

👉 目标不是赚钱，而是验证：
	•	用户会不会反复用
	•	哪种输入最多（拍照 / 外卖）
