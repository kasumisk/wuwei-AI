# 四、AI筛选要求

请实现 AI Vision Filter 服务。

输入：
- food name
- aliases
- category
- cuisine
- food form: ingredient / dish / semi_prepared
- candidate image urls

输出 JSON：

{
  "bestCandidateIndex": 0,
  "candidates": [
    {
      "index": 0,
      "isFoodImage": true,
      "isMatchedFood": true,
      "matchScore": 0-100,
      "qualityScore": 0-100,
      "aestheticScore": 0-100,
      "safetyScore": 0-100,
      "finalScore": 0-100,
      "reason": "..."
    }
  ],
  "decision": "select | reject_all | need_manual_review",
  "reason": "..."
}

评分规则：

- matchScore：是否真的是目标食物
- qualityScore：清晰度、分辨率、曝光、无模糊
- aestheticScore：是否适合作为 App 主图
- safetyScore：是否无水印、无人物脸、无品牌侵权、无恶心/不适画面
- finalScore = matchScore * 0.45 + qualityScore * 0.25 + aestheticScore * 0.2 + safetyScore * 0.1

选择规则：

- finalScore >= 80：自动选择
- 60 <= finalScore < 80：进入人工审核
- < 60：拒绝
- 如果 matchScore < 70，即使 finalScore 高也不能自动选择
- 如果 safetyScore < 80，不能自动选择
