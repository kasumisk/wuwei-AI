Create a clean and highly readable Lottie animation for a website hero banner of an AI nutrition app called "EatCheck".


This is a PRODUCT UI animation, not a decorative animation.

Core concept:
"Scan food → Analyze nutrition → Decide if it's suitable to eat"

IMPORTANT:
- Prioritize clarity, readability, and strong visual focus
- Avoid heavy blur, fog, or excessive gradients
- Main UI must be clearly visible even at small size

Layout:
- Animation centered or slightly right-aligned
- Leave space on the left for headline text
- One clear focal point at all times

Scene flow:

1. Food Input (Start)
- A simple food item appears (plate with egg, salad, meat OR a generic food icon)
- Clean, minimal vector style
- A scanning line or camera frame passes over the food
- Keep it simple (no complex illustration)

2. Recording / Input
- The food smoothly transforms into a simple input card
- Small UI elements appear:
  - food name
  - simple icons
- Motion is smooth and calm

3. AI Analysis (Core Moment)
- The food or input card transitions into 3 clear nutrient indicators:
  - Protein
  - Fat
  - Carbs
- Use simple bars or circular progress
- Keep spacing wide and readable
- No clutter, max 3–4 elements

4. Decision Result (FOCUS)
- A clean, solid UI card appears (MAIN FOCUS)
- High contrast (dark text on light background)

Example content:
- "Good to eat" (green checkmark)
OR
- "Not ideal now" (soft warning icon)

- Optional subtext:
  "Protein low" / "Fat balanced"

IMPORTANT:
- This card must be the most visible element
- No transparency or blur on this card

5. Feedback Animation
- Subtle glow or pulse around result
- Calm confirmation feeling

Animation style:
- Smooth, slow, confident motion
- No fast transitions
- Loop seamlessly (4–6 seconds)

Design style:
- Minimal SaaS (Stripe / Apple inspired)
- Flat + slight depth (very light shadow only)
- No heavy glassmorphism

Color:
- Background: very light (almost white)
- Primary accent: green (#22C55E)
- Neutral UI: white / gray
- Ensure strong contrast

Typography:
- Large, clear, readable
- No tiny or faded text

DO NOT:
- Do not blur the main UI
- Do not use heavy gradients behind text
- Do not overload with particles or effects

Test requirement:
The animation must remain clear and readable when scaled down to 50% and placed behind website text.

Output:
- Lottie JSON
- Optimized for web performance
- Lightweight vector shapes only
