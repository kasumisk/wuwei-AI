
## ✅ 1. 标准调用方式（统一写法）

例如：

```ts
this.i18n.t('recommendation.low_protein', {
  lang,
  args: { value: 20 }
})
```

---

## ✅ 2. AI Prompt 改造方案（非常关键）

改造成：

```ts
const prompt = this.promptBuilder.build('diet_analysis', {
  lang,
  data
})
```

并提供：

* prompt 多语言模板结构
* 示例（en / zh）

---

## ✅ 3. 中间件 / 基建优化（强制）

必须设计：

### 🌍 全局 LanguageContext

```ts

  getLang(): string
}
```

避免手动传递丢失

---

### 🔗 i18n Guard / Interceptor

自动：

* 从 header / user profile 读取 lang
* 注入 request
